# cologneWeather
Weather Data from Cologne, Germany

# Dependencies
Install dependencies via `pip install -r requirements.txt`

 Github help
 TO PUSH TO GITHUB
 1.) `git status` (to check what's up)
 2.) `git add <filename> or .` (for everything)
 3.) `git commit -m "<your commit message>"`
 4.) `git push`

 TO UPDATE FROM REPO
 1.) `git status`
 2.) `git pull`
